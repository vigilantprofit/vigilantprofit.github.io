# Vigilant Profit Solutions Website

Complete GitHub Pages site with mobile responsive design.

## Files Included:
- index.html (homepage)
- ai-automation.html
- custom-software.html  
- cloud-transformation.html
- enterprise-solutions.html
- payments.html
- about.html
- styles.css

## To Deploy:
1. Create logo.png (200x100px, transparent background)
2. Upload all files to GitHub repository
3. Enable GitHub Pages in Settings
4. Site will be live at yourusername.github.io

## Logo:
Place a logo.png file (60-96px height) in the root directory.
If you don't have one yet, you can use a text placeholder or create one at canva.com
